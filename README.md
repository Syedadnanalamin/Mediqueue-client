# MediQueue

A tutor booking platform where students can find tutors, book sessions, and tutors can manage their tutorial services.

## Live Site

https://your-live-site.vercel.app

## Features

* Secure login and registration with Google and Email authentication.
* Browse available tutors with detailed information.
* Add, update, and delete tutor tutorials.
* Book tutor sessions and manage booked tutors.
* Responsive design for mobile, tablet, and desktop devices.
